Opis szczegółowy

Klasa reprezntująca menu gry.

Klasa będąca menu gry jest jednocześnie jednym ze stanów gry (dziedziczy z BaseState). Jest pierwszym stanem jaki wrzucany jest do menadżera stanów po uruchomieniu aplikacji i jednoczesnie ostatnim zdejmowanym ze stostu stanem. Dziedziczy również z GUIGroup by umożliwić sprawną manipulację elementami graficznymi menu.
