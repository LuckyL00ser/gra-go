Opis szczegółowy

Klasa reprezentująca stan aplikacji typu gra.

Dziedziczy z klasy BaseState. Łączy logikę gry z warstwą prezentacji (wyłącznie dla stanu rozgrywki), agregując obiekty klas Game oraz GameGUIView.
