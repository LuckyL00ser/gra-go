Opis szczegółowy

Obiekt tej klasy zarządza gniazdem TcpSocket odpowiadającym za komunikację po protokole TCP.

)
virtual

Metoda inicjuje połączenie TCP do serwra.

Metoda podejmuje próbę połączenia się z serwerem TCP gry o adresie IP zdefiniowanym w ustawieniach gry. Łączenie się poprzez gniazdo TCP jest operacją blokującą dlatego zaleca się wywołanie tej metody w osobnym wątku. Funcka zwraca prawde w przypadku ustanowienia połączenia. Fałsz gdy nie można się połączyć, bądź przekroczono systemowy czas połączenia.

Parametry
portPort do którego łączymy się.
ipIP serwera do którego łączymy się.
Zwraca
bool Zwraca prawdę w przypadku pozytywnego rezultatu połączenia.

Reimplementowana w TcpServer.
